```

├── prompts/            # 可复用 prompt 片段

├── docs/
│   ├── topo-kb-meta/      # 从 14-KB 迁入的 17 个项目管理文件
│   └── specs/             # 设计文档 × 2
└── plans/             # 实现计划 × 2
```

### 核心设计

- **5 个原子步骤**：search → filter → analyze → correlate → deposit，按需组合
- **3 种模式**：全流程（thesis）、兴趣阅读（interest）、快速扫描
- **TEX-first**：arXiv e-print → curl → extract → read .tex；WebFetch 不可用
- **priority 分流**：thesis → for-thesis/，core → papers/ + indexes/，interest → daily/ + indexes/
