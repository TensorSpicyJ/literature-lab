# Daily Radar Prompt

Produce today's `paper-radar-YYYY-MM-DD.md` and deposit it to the literature library.

## Output Location
`D:\BaiduSyncdisk\code\OB_NOTE\MY NOTE\12-每日文献知识库\daily\{year}-{month}\paper-radar-{today}.md`

## Structure

Follow `protocols/feishu-format.md` exactly:

```
# Paper Radar — {today}

## 今日结论

## 必看论文

## 背景 / 前沿跟踪

## 对 thesis 最直接的用处

## 当日沉淀

```

## Rules
- Published/accepted papers in 必看论文; arXiv/preprint in 背景/前沿跟踪
- Each 必看论文 entry: **一句话** / **状态** / **为什么重要** / **和 TS 的关系**
- If no qualifying papers: state honestly in 今日结论
- 当日沉淀 lists actual files created
