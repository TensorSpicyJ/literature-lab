# Strange Metal and Fermi Arcs from Disordering Spin Stripes

> Zhang, Bultinck (2026) — Ghent Univ. — [arXiv:2507.06309](https://arxiv.org/abs/2507.06309)
> **来源雷达:** [[daily/2026-05/paper-radar-2026-05-05]]

## 一句话
将自旋条纹物理连接到 Patel et al. (Science 2023) 的**奇异金属普适理论**：电势无序 + 自旋条纹 → SDW 玻璃态 → 涌现 Ising 自旋玻璃 → QCP 由空间随机电子-玻色子耦合模型描述 → 自然地产生线性-T 电阻率和费米弧。

## 核心主张
1. **自旋条纹无序化是奇异金属的微观起源**——电势无序将 clean 模型的 O(2) QCP 转变为强 disordered SDW 玻璃
2. **SDW 玻璃态的 Ising 变量与 Patel 理论精确对应**——耦合常数 g(r) = g S_r cos(Qr_x − α_r)，⟨g⟩=0 自然满足
3. **费米弧不需要长程 AFM 序**——Monte Carlo 显示 ξ ~ 4-5 个晶格常数即产生清晰 Fermi arcs
4. **热涨落的 Ising 变量自平均化 g(r) 的符号**——有限温度下 sign 自动平均
5. **电荷槽（charge trough）作为 AFM domain wall**——理解赝能隙区空间结构的微观图像

## 模型

清洁极限有效作用量:
$$S_{\text{clean}} = S_\psi + S_\Phi$$

$$S_\psi = \int d\tau \sum_k \bar{\psi}(\partial_\tau + \varepsilon_k)\psi + g \int d\tau \sum_r (-1)^{r_x+r_y} l^a(r) \Phi_{aj}(r) \bar{\psi}(r) \sigma^j \psi(r)$$

序参量流形: (S¹ × S²)/ℤ₂ → 半涡旋 + Ising 自由度

无序项: S_dis = S_clean − ∫ dτ Σ μ(r) ψ̄ψ → S_dis^Φ = −∫ dτ Σ (w^a Φ_{aj})²

## 对称性约束 / 关键机制

**相图路径（clean → disordered）：**
```
SDW (J大) → [O(2) QCP] → 自旋向列相 (nematic, Q_{jl}≠0, ⟨Φ⟩=0) → [O(3) QCP] → 对称相
```

**无序效应：**
- 随机场 pinning CDW → θ_r = α_r + (1−η_r)π/2, η_r=±1
- 向列相刚性 → m_r = N_r s_r, s_r=±1
- Ising 自旋玻璃: H_eff = −Σ_{⟨rr'⟩} J_{r,r'} S_r S_{r'} （J_{r,r'}=J cos(α_r−α_{r'}), ⟨J⟩=0）
- 横向场 h_r ~ exp(−4√J|w(r)|/v) → 指数衰减 → 零温 Ising 玻璃序
- QCP: O(3) 临界场 N 耦合 Fermi 面 → **Patel 普适理论**

## 推演链

```
自旋条纹 SDW (Q, M) → 电-声耦合 → CDW ρ_{2Q} ∝ S_Q·S_Q
→ 序参量 Φ (2×3 实矩阵) → 流形 (S¹×S²)/Z₂
→ J 减小 → O(2) disorder → 自旋向列 (Q_{jl}=⟨m_j m_l⟩−δ_{jl}/3 ≠ 0)
→ 电势无序 → CDW pinning → θ_r 锁定
→ |w|≫J → η_r, s_r 涌现 → S_r = s_r η_r (gauge invariant)
→ Ising 玻璃 H ~ −Σ J_{r,r'} S_r S_{r'} → 零温有序
→ O(3) QCP: 有效理论含随机耦合 g(r) = g S_r cos(Qr_x−α_r)
→ ⟨g⟩=0 (自旋玻璃) → Planck 散射率 → 线性-T 电阻率
→ 热涨落 → S_r 自平均 → 有限 T 稳定非费米液体
```

## 完整推演 {#完整推演}

### 1 — Clean 模型相图

序参量: SDW S_Q 的 Fourier 分量 → 2×3 实矩阵 Φ → (S¹×S²)/ℤ₂ 流形。

Landau-Ginzburg 势: −r Tr(ΦᵀΦ) + (u/4)(Tr(ΦᵀΦ))² − Σ_i (v/4)(Tr(ΦᵀτⁱΦ))²
→ 极小值: Φ_{aj} = |Φ| n_a m_j（rank-1）

J̃ 项 → 自旋向列序: Q_{jl} = ⟨m_j m_l⟩ − δ_{jl}/3

相图: SDW → O(2) QCP → 自旋向列 → O(3) QCP → 对称相

### 2 — 无序 → Ising 自旋玻璃

电势无序 μ(r) → 对 Φ 的随机场项 −∫ (w^a Φ_{aj})²

向列相: |w| ≫ J → θ_r pinning → η_r
m_r pinning → s_r
S_r = s_r η_r (ℤ₂ gauge invariant)

Ising 自旋玻璃: H_eff ~ −Σ J_{r,r'} S_r S_{r'}，⟨J⟩=0

半涡旋（half-vortex）→ 如果引入可解除 frustration，但向列刚性（J̃ 大）使得域壁比涡旋便宜。

### 3 — O(3) QCP → Patel 奇异金属理论

有效作用量——随机耦合:
S = S_ψ − Σ μ(r)ψ̄ψ + Σ g(r)(−1)^{r_x+r_y} N·ψ̄σψ + J̃(∂N)²

g(r) = g S_r cos(Qr_x − α_r)

⟨g(r)⟩ = 0（自旋玻璃）→ Patel et al. (2023): 此条件下散射率达到 Planck 极限，产生线性-T 电阻率。

### 4 — 费米弧的 Monte Carlo

Plaquette-Ising 模型 (Eq. 21):
H = −t Σ c†_r c_{r'} − t' Σ c†_r c_{r'} − μ Σ c†c + g Σ (−1)^{r_x+r_y} S_{R(r)} c†σ^z c − J Σ S_R S_{R'}

关键参数: t=1, t'=−0.35, μ=1.4, g=1, J=0.1

结果: T_c ~ 0.205, ξ(T=0.254) ~ 4.18 (L=40)
零频谱权重: nodal 方向亮, anti-nodal 被压制 ~7.2×
→ 短程 AFM 即可产生 Fermi arcs

RKKY 分析 (Appendix C): plaquette coupling 引入竞争 FM/AFM 贡献 → 降低 T_c（与普通 on-site 耦合相反）
