# Anisotropic Superconducting Diode Effect in Planar Josephson Junctions

> Chilampankunnel Prasannan, Pekerten, Alashkar, Matos-Abiague (2026) — Wayne State / SUNY Buffalo — [arXiv:2604.17594](https://arxiv.org/abs/2604.17594)
> **来源雷达:** [[daily/2026-05/paper-radar-2026-05-05]]

## 一句话
系统理论研究了 Rashba + Dresselhaus SOC 共存下 planar Josephson 结中 SDE 的**磁和晶各向异性**，给出 SDE 抑制的对称性条件（可实验检验），并预测栅压可在纯 Rashba 下实现低场极性反转。

## 核心主张
1. **SDE 抑制的对称性条件（Table I）：** 5 类条件下 SDE 消失，仅由 SOC 对称性 + B 场方向 + 结取向决定，与场强无关
2. **Dresselhaus SOC 引入晶各向异性：** 纯 Rashba → SDE 仅磁各向异性；纯 Dresselhaus → 显著晶各向异性（二重对称）
3. **栅压诱导低场极性反转：** 即使在纯 Rashba 下，调节结势 $V_0$ 可使 I_c+ 和 I_c- 在低场交叉（非 0-π 转变）
4. **窄结解析模型：** η ∝ (1-τ)^{1/4} (k_so/k_F) φ_q，Cooper 对动量 $\tilde{q}$ 编码各向异性
5. **SOC 比值可从 SDE 零线提取：** α/β = cot θ_B* cos 2θ_C - sin 2θ_C

## 模型
BdG 哈密顿 in 2DEG planar JJ:

$$H = \sigma_z \otimes (H_0 + H_{SO}) + H_Z + \Delta(x,\phi)\tau_+ + \Delta^*(x,\phi)\tau_-$$

$$H_R = \frac{\alpha}{\hbar}(p_y\sigma_x - p_x\sigma_y), \quad H_D = \frac{\beta}{\hbar}[(p_x\sigma_x - p_y\sigma_y)\cos 2\theta_C - (p_x\sigma_y + p_y\sigma_x)\sin 2\theta_C]$$

SOC 参数化: α = λ cos θ_so, β = λ sin θ_so

## 对称性约束 / 关键机制

**SDE 抑制条件 (Table I):**

| 条件 | SOC | 约束 |
|------|-----|------|
| (1) | α≠0, β=0 | sin θ_B = 0 |
| (2) | α=0, β≠0 | cos(2θ_C + θ_B) = 0 |
| (3) | α≠0, β≠0, β≠α | cos 2θ_C = sin θ_B = 0 |
| (4) | β=±α | cos(2θ_C+θ_B) = ±sin θ_B |
| (5) | β=±α (gauge) | sin(2θ_C+θ_B) = ∓cos θ_B |

**现象学模型核心：**
$$I \approx I_1 \sin(\phi-\phi_1) + I_2 \sin(2\phi-\phi_2)$$
$$\eta \approx -2\sin(2\phi_2-\phi_1) \propto \mathbf{w}_x \cdot \mathbf{B}$$

仅在至少两个谐波共存时 phase-unbiased SDE 非零。

## 推演链

```
BdG 哈密顿 → 自由能 F(B,φ) → SDE 条件: F(B,φ) ≠ F(B,-φ)
→ 对称性变换 U = R_x^k R_y^l [Γ(u)]^m T^n → 若 U⁻¹H(B,φ)U = H(B,-φ) → SDE 抑制
→ 展开 a_m ∝ w_x·B（一阶） → 抑制条件 w_x·B = 0
→ 窄结解析: δ-势近似 → ABS 能谱 → 自旋劈裂费米面 k_{F±} → η ∝ q̃
→ Ω_± 编码各向异性: Ω_± = √(1 ± sin 2θ_so sin 2θ_C)
→ k̃_so ∝ Ω_-, q̃ ∝ Ω_+^{-1}
```

## 完整推演 {#完整推演}

### 1 — BdG 哈密顿量

2DEG 上的 planar JJ，包含 Rashba + Dresselhaus SOC、面内 Zeeman 场、SC 配对势。θ_C 为电流方向与 [100] 晶轴的夹角，θ_B 为 B 场方向与电流的夹角。

### 2 — 对称性分析

寻找使 H(B,φ) 与 H(B,-φ) 等价的变换 U。U = R_x^k R_y^l [Γ(u)]^m T^n （k,l,m,n ∈ {0,1}）。若存在 → SDE 抑制。结果如 Table I。

特殊情况 β=±α: 可通过 U = exp(i q·r [σ₀⊗(v·σ)]) 完全 gauge away SOC → SDE 消失。条件 (5) 的零线不伴随 η 符号变化（非对称性保护，而是 gauge 消除）。

### 3 — 现象学模型

F(B,φ) = F₀ + Σ[a_m(B)sin(mφ) + b_m(B)cos(mφ)]

a_m (odd in B) 是 SDE 的来源。最低阶:
a_m ≈ f_m (w_x·B), b_m ≈ g_m

w_x = (β cos 2θ_C, -β sin 2θ_C - α, 0)ᵀ

→ SDE 条件: w_x·B = β cos(2θ_C+θ_B) - α sin θ_B ≠ 0

### 4 — 窄结解析（δ-势近似）

W_N ≪ ξ ≪ W_S → V(x) ≈ V₀W_N δ(x), W_S → ∞

Rashba-only, k_y=0: SOC 场沿 y → 仅 B_y 有效

ABS 能谱 → F ≈ (L/π) Σ E_{νσ}^<(k_y=0) k_{F±}

k_{F±} = k_F ± k_so, k_so = m*α/ℏ²

η ≈ (1-τ)^{1/4} (k_so/k_F) q W_N

q ≈ κ_B²/k_F sin θ_B（Cooper 对动量，Rashba-only）

### 5 — Rashba + Dresselhaus 推广

U_θ 旋转 → 等效纯 Rashba 系统:
α̃ = λ Ω_+, k̃_so ≈ k_λ Ω_-
q̃ ≈ κ_B² [sin θ_B cos θ_so - sin θ_so cos(2θ_C+θ_B)] / (k_F² Ω_+)

θ_C=π/4 → η(θ_B) ~ sinusoidal（与 Schiela2025 实验一致）

θ_B 零线 → α/β = cot θ_B* cos 2θ_C - sin 2θ_C（可实验提取 SOC 比值）
