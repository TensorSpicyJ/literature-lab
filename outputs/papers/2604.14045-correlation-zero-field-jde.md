# Strong Correlation Drives Zero-Field Josephson Diode Effect

> Sun, Zhang, He (2026) — USTC Hefei, ICQD / Hefei National Lab — [arXiv:2604.14045](https://arxiv.org/abs/2604.14045)
> **来源雷达:** [[daily/2026-05/paper-radar-2026-05-05]]

## 一句话
在 Hubbard U 强关联 + 奇数电子数的 Josephson 结中，电子-电子相互作用自发破缺时间反演和镜面对称性，形成 φ-junction，实现**无需磁序的零场 Josephson 二极管效应**。

## 核心主张
1. **强关联是零场 JDE 的独立机制**——不依赖 SOC、磁序或 explicit symmetry breaking
2. **奇数电子数 + 破缺 SU(2) 对称性**是 φ-junction 的必要条件；SOC 仅破 SU(2)，不决定二极管极性
3. **φ-junction 的简并基态** (±φ) 使临界电流不对称，I_c+ ≠ I_c-
4. **外加微小 Zeeman 场可大幅增强 JDE**——磁性关联放大效应，效率在能级交叉处达峰值
5. **极性非 SOC 决定**——η 是 SOC 强度 λ 的偶函数，与磁手性机制形成鲜明对比
6. 有限温度下 (T ≪ Δ)，上能级参与使得自由能光滑化，但对零场和低场 JDE 定性结论不变

## 模型
Hubbard 模型 on square lattice + Rashba SOC + 超导 proximity:

$$H = \sum_{<ij>} [t \sum_{\sigma} c^\dagger_{i\sigma} c_{j\sigma} + \Delta_i c^\dagger_{i\uparrow} c^\dagger_{j\downarrow} + h.c. - \mu n_i] + \sum_i U_i n_{i\uparrow} n_{i\downarrow} + H_{\mathrm{SOC}}$$

- 正常区用 minimal 4-site 模型，SC 导联通过 pairing 项 Δ_i（相位 0 或 φ）纳入
- 精确对角化求解
- RCSJ 模型分析相位动力学

## 对称性约束 / 关键机制

对称性分析 (Table I):
- SU(2)、M_y、M_z、C₂T、PT 导致基态简并 E₀(φ) = E₁(φ) → **必须全部破缺**
- 非均匀 Hubbard U（如图 1a）破缺上述对称性
- M_x T 保护 ⟨S_x⟩ = 0

**与扭曲 d-波双层对比：**
- d-wave bilayer φ-junction 来自 harmonic interference（二阶 Josephson 隧道主导）
- 本机制来自强关联 + 自旋极化 + 简并度解除
- U=0 → 常规结 (Kramers 确保偶数电子数)

## 推演链

```
强关联 → AFM 自旋关联 → 总自旋 S=1/2 (奇数电子数)
→ 基态在 φ=0/π 处 Kramer 简并 (TR 保护)
→ φ偏离0/π → TR破缺 → 简并解除 → 能级交叉
→ φ-junction (±φ 简并基态)
→ RCSJ: 不对称势垒高度 → I_{c+} ≠ I_{c-} → 零场 JDE
→ 施加 B_z → 塞曼能推动能级 → 耦合产生 avoided crossing
→ |⟨S_z⟩| > 0 态下移，⟨S_z⟩ < 0 态上移
→ 临界 B_p 处效率 η 达峰值，B_p ~ Δ²
```

## 完整推演 {#完整推演}

### 1 — 模型设定

Hubbard 模型描述 Josephson 结正常区。半无限 SC 导联通过配对项 Δ_i（携带相位 0 或 φ）纳入有效哈密顿量——此 treatment 由 Affleck2000PRB 和 Bergeret2007PRB 建立。正常区仅 4 个格点，用精确对角化求解。

### 2 — 偶数 vs 奇数电子数

**N_e 偶数：** 总自旋整数，TR 保持下基态不简并。SC 配对后 E₀ = -E_J cos φ ——常规 Josephson 关系。

**N_e 奇数：** 总自旋半整数，TR 对称性要求 φ=0 或 π 处基态二重简并 (Kramers)。φ 偏离 0/π → TR 破缺 → 简并解除 → 最低能量出现在 ±φ ≠ 0,π → φ-junction。

### 3 — 对称性分析

哪些对称性导致 E₀(φ) = E₁(φ)（即阻止 φ-junction）？
- SU(2): 自旋简并 → 无 φ-junction
- M_y, M_z: 自旋 S_± → S_∓ → 二维表示 → 简并
- C₂T, PT: C₂(或P) 翻转 φ→-φ, T 再翻转回来 → (C₂T)² = -1 → Kramers 型简并
- **M_x T**: 翻转 S_x 但不导致简并 → **可保留**

非均匀 U 分布破缺 M_y, M_z, C₂T, PT → φ-junction 条件满足。

### 4 — 自旋极化与 SEE 的差异

SEE (超流 Edelstein 效应): SOC 直接极化自旋，极化随 λ 连续增长。
φ-junction: ⟨S_z⟩ 在 λ→0 仍有限 → SOC 不直接极化，仅破 SU(2)。

### 5 — Zeeman 场效应

$$H_Z = -\sum_i \mathbf{B} \cdot \boldsymbol{\sigma} c^\dagger_{i\alpha} c_{i\beta}$$

- 小 B_z: 耦合 |S_z>↑ 和 |S_z>↓ → avoided crossing → I_s(φ) 有两个局部极小值
- B_z 增大: 两个极小值靠近 → 全局极小在 B_p 处切换 → **η 峰值**
- B_z > B_p: spin locking → η 衰减
- B_p ~ Δ² (图 4a 虚线)
- η(λ) 为偶函数 (图 4b 插图)——SOC 不决定极性
