# Pseudogap and Condensation in Cuprate Superconductors from NMR Shifts

> Lee, Haase (2026) — Univ. Leipzig, Felix Bloch Institute — [arXiv:2604.19215](https://arxiv.org/abs/2604.19215)
> **来源雷达:** [[daily/2026-05/paper-radar-2026-05-05]]

## 一句话
重新分析 ~40 种铜基超导体的 planar Cu 和 O 的 NMR 位移数据，揭示**两个独立自旋分量**（各向异性 A 自旋 + 各向同性 B 自旋）的掺杂和温度演化，确定赝能隙在 NMR 位移中的标志性特征，并发现三种凝聚（超导转变）斜率模式。

## 核心主张
1. **Cu 位移可严格分解为两个独立自旋分量**——A (3d 各向异性, A_∥ 为负) 和 B (4s 各向同性)，无需假设超精细常数具体数值
2. **B 自旋 DOS 随掺杂线性下降**——下降率在 x≈0.20 上下有三倍差异（Δ_x χ_B per Δx: 0.61 below vs 1.65 above 最佳掺杂）
3. **赝能隙 = Tc 处位移的展宽**（非弛豫展宽）——随着掺杂降低，展宽向 Tc 以上延伸
4. **凝聚有三种斜率模式：** Δχ_A=0+Δχ_B≠0; |A_∥|Δχ_A = BΔχ_B; |A_∥|Δχ_A ≈ 0.5BΔχ_B
5. **最优 Tc 对应第三种斜率**（A 和 B 以 1:2 比例同时凝聚），且需赝能隙存在
6. **Tc_max 不直接编码于位移**——而是编码于 Cu 弛豫各向异性和 planar O 空穴含量

## 模型
超精细耦合分解 (Mila 1989, Pavarini 2001):

$${^{63}K}_\perp(T) = B \chi_B(T) + A_\perp \chi_A(T)$$
$${^{63}K}_\parallel(T) = B \chi_B(T) + A_\parallel \chi_A(T)$$

其中 |A_∥|/A_⊥ ≈ 6, A_∥ < 0。

Planar O 位移仅耦合 A 自旋: ¹⁷K_c(T) = C χ_A(T)

## 推演链

```
Cu 位移(∥,⊥) → 超精细分解 → χ_A(T) + χ_B(T) 独立分量
→ 高掺杂: 两者均金属性（T 无关平台 + Tc 处尖锐下降）
→ 降低掺杂: B-spin DOS 先快速下降（dome 上边缘），后缓慢下降（dome 内）
→ 赝能隙出现: Tc 处展宽 + 向 Tc 以上延伸
→ χ_B vs χ_A (T 隐参数): 三条斜率线
  - Slope 1: Δχ_A=0, Δχ_B≠0 （过掺杂，B 主导凝聚）
  - Slope 2: |A_∥|Δχ_A = B Δχ_B （ΔK_∥=0, 全部 LSCO + 欠掺杂 YBCO）
  - Slope 3: |A_∥|Δχ_A ≈ 0.5 B Δχ_B （最优掺杂 + 高 Tc 材料）
→ χ_B(300K) vs T_c: 画出超导 dome
→ T_c,max ~ 200K × 2n_p (planar O hole content)
```

## 完整推演 {#完整推演}

### 1 — 历史背景：为什么需要新分解

早期 NMR（1988-89）：Barrett, Takigawa, Walstedt 等在 YBCO7 上发现 Tc 下方 shift 急剧下降（d-wave 配对 + Yosida 函数）。但后续数据来自更多铜基体系后，single-spin 模型无法解释 Cu(∥) 和 Cu(⊥) 不缩放的事实。

### 2 — 双超精细耦合分解

核心假设：Cu 有两个独立的超精细耦合——A_α (3d x²-y² 轨道, onsite, 各向异性, A_∥ 负) 和 B (4s, 各向同性, 通过 O 转移)。不做 B = 4B' 的单带假设。

Cu(⊥) ≈ Bχ_B（因为 A_⊥ 很小）
Cu(∥) = Bχ_B + A_∥χ_A（A_∥ 负值 → 降低 ∥ shift）

O(c) 仅耦合 χ_A（无 isotropic 项）

### 3 — 掺杂演化

关键发现：Bχ_B(300K) 随 doping 下降——从 heavily OD 到 optimal doping 下降很快（rate ~1.65），从 optimal doping 到 UD 下降缓慢（rate ~0.61）。χ_A 基本不变（微弱 family 依赖）。

### 4 — 赝能隙的 NMR 标记

Tc 处的 "broadening"——不同于弛豫数据中始终 sharp 的 Tc 转变。展宽随着降低 doping 向 Tc 以上延伸——这就是 NMR 赝能隙。赝能隙能量标度可从 O 位移高温变化中定量提取。

### 5 — 凝聚三斜率

χ_B vs χ_A（T 隐参数）图中：
- Slope 1 (Δχ_A=0): B 主导凝聚，过掺杂特征
- Slope 2 (|A_∥|Δχ_A = BΔχ_B): A 和 B 等比例凝聚 → ΔK_∥ = 0。全部 LSCO、YBCO、YBCO8 有此行为
- Slope 3 (0.5×B): 最优掺杂 + 最高 Tc 材料。自高温金属态到凝聚态全程遵循此斜率。Tc 以上已有配对（赝能隙区），Tc 处仅建立相位相干

### 6 — 与弛豫数据的关系

B-spin DOS 变化解释 Cu 弛豫各向异性（Lee2026 / 2604.10133）。弛豫各向异性 ~3.6 (UD) → ~1 (high OD)，与 T_c 相关。

Tc_max 不编码于 shift 而编码于弛豫 + planar O hole content: T_c,max ≈ 200K × 2n_p（Rybicki2016, DMFT 验证 Kowalski2021）。
